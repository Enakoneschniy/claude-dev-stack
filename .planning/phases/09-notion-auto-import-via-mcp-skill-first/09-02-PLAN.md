<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user CLI args -> notion-cli.mjs | User-supplied URL and flags passed to add/import subcommands |
| claude mcp list output -> doctor | External CLI output parsed as JSON; could be malformed |
| MCP tool invocation -> skill | Skill instructs Claude to call MCP tools; auth managed by Claude runtime |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-07 | Spoofing | notion add URL arg | mitigate | parseNotionUrl validates URL structure; only 32-hex page_id extracted; no arbitrary string accepted |
| T-09-08 | Tampering | claude mcp list --json | mitigate | JSON.parse in try/catch; malformed output triggers warn, not crash |
| T-09-09 | Information Disclosure | notion list output | accept | Displays page URLs which user already configured; no secrets exposed |
| T-09-10 | Denial of Service | doctor MCP check | mitigate | execFileSync timeout set to 10000ms; failure is non-fatal warning |
| T-09-11 | Elevation of Privilege | skill MCP invocation | accept | MCP tools run within Claude session's existing auth scope; no privilege escalation possible |
</threat_model>
