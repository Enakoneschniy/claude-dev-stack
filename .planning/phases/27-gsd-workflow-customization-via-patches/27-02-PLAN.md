<objective>
Add a regression test that locks the wizard's behavior of copying shipped
`patches/` to `~/.claude/gsd-local-patches/`. This is the step that pins the
authoritative patch source consulted by `gsd-auto-reapply-patches.sh` —
without this copy, the auto-reapply hook falls through to dev-checkout paths
that may not exist on end-user machines.

Purpose: Formalize the patches mechanism — a future refactor of the install
wizard that accidentally removes the `cpSync(patchesSrc, patchesDest)` block
would silently break `/gsd-update` survival. This test catches that regression.

Output: One new test file under `tests/`.
</objective>

<interfaces>
`installSessionHook(stepNum, totalSteps, pkgRoot, vaultPath, projectsData)` is
exported from `lib/install/hooks.mjs`. It:

1. Ensures `~/.claude/hooks/` exists.
2. Copies `<pkgRoot>/patches/` recursively to `~/.claude/gsd-local-patches/`
   if `<pkgRoot>/patches/` exists (BUG-06 D-07 block, lines 16–24).
3. Copies several hook scripts to `~/.claude/hooks/`.
4. Writes per-project settings.json if `projectsData.projects` is non-empty,
   otherwise writes a global fallback settings.json.

The function uses `homedir()` from `node:os` — overriding via `HOME` env
variable controls its destination on Unix. The test must set `HOME` before
importing the module (or use `child_process` to run the function in a fresh
process). Simpler: run via `child_process` with HOME override.

Test harness notes:
- Existing tests import library modules directly and set up fixtures in tmp dirs.
- For HOME override at module load time, spawn a child node process with
  `--input-type=module -e "import { installSessionHook } from '<abs>';
  installSessionHook(1, 1, '<pkgRoot>', '<vault>', { projects: [] })"`, with
  `env: { HOME: fakeHome, PATH: ..., ... }`.
- Or: use `process.env.HOME = fakeHome` before `await import(...)` — works
  because `os.homedir()` re-reads `HOME` on Unix at call time, not at import.

Recommend the second approach (simpler, same process) but validate that
`homedir()` actually re-reads HOME. Per Node docs on Unix it reads
`$HOME` then falls back to `/etc/passwd`. For macOS/Linux CI it re-evaluates
per call. Use this directly.
</interfaces>
</context>

       ```js
       import { describe, it, before, after } from 'node:test';
       import assert from 'node:assert/strict';
       import { existsSync, readFileSync, writeFileSync, mkdirSync, mkdtempSync, rmSync } from 'fs';
       import { join, dirname } from 'path';
       import { fileURLToPath } from 'url';
       import { tmpdir } from 'os';

       const __dirname = dirname(fileURLToPath(import.meta.url));
       const repoRoot = join(__dirname, '..');

       describe('installSessionHook copies patches/ to ~/.claude/gsd-local-patches/', () => {
         let fakeHome;
         let fakePkgRoot;
         let origHome;
         let installSessionHook;

         before(async () => {
           fakeHome = mkdtempSync(join(tmpdir(), 'cds-home-'));
           fakePkgRoot = mkdtempSync(join(tmpdir(), 'cds-pkg-'));

           // Seed a minimal pkgRoot/patches/ tree
           mkdirSync(join(fakePkgRoot, 'patches'), { recursive: true });
           writeFileSync(join(fakePkgRoot, 'patches', 'manager.md'), 'manager patch body\n');
           writeFileSync(join(fakePkgRoot, 'patches', 'transition.md'), 'transition patch body\n');

           // Seed a minimal pkgRoot/hooks/ tree — installSessionHook tries to copy several scripts
           mkdirSync(join(fakePkgRoot, 'hooks'), { recursive: true });
           // Provide empty stubs for every script name the function tries to copy
           for (const name of [
             'session-start-context.sh', 'session-end-check.sh', 'vault-auto-push.sh',
             'gsd-auto-reapply-patches.sh', 'budget-check.mjs', 'budget-reset.mjs',
             'budget-check-status.mjs', 'gsd-workflow-enforcer.mjs',
             'notebooklm-sync-trigger.mjs', 'notebooklm-sync-runner.mjs', 'update-context.mjs',
           ]) {
             writeFileSync(join(fakePkgRoot, 'hooks', name), '#!/bin/bash\nexit 0\n');
           }

           // Seed a minimal lib/budget.mjs so the budget copy block succeeds
           mkdirSync(join(fakePkgRoot, 'lib'), { recursive: true });
           writeFileSync(join(fakePkgRoot, 'lib', 'budget.mjs'), 'export default {};\n');

           // Override HOME, then import (hooks.mjs calls homedir() per invocation)
           origHome = process.env.HOME;
           process.env.HOME = fakeHome;

           ({ installSessionHook } = await import('../lib/install/hooks.mjs'));
         });

         after(() => {
           if (origHome === undefined) delete process.env.HOME;
           else process.env.HOME = origHome;
           rmSync(fakeHome, { recursive: true, force: true });
           rmSync(fakePkgRoot, { recursive: true, force: true });
         });

         it('copies patches/*.md to <HOME>/.claude/gsd-local-patches/', () => {
           // Call the function — passing empty projectsData triggers fallback branch;
           // patches copy must still happen because it is unconditional.
           installSessionHook(1, 1, fakePkgRoot, join(fakeHome, 'vault'), { projects: [] });

           const destDir = join(fakeHome, '.claude', 'gsd-local-patches');
           assert.ok(existsSync(destDir), 'destination dir must exist');
           assert.ok(existsSync(join(destDir, 'manager.md')), 'manager.md must be copied');
           assert.ok(existsSync(join(destDir, 'transition.md')), 'transition.md must be copied');

           assert.equal(
             readFileSync(join(destDir, 'manager.md'), 'utf8'),
             'manager patch body\n',
             'manager.md content must match source byte-for-byte',
           );
           assert.equal(
             readFileSync(join(destDir, 'transition.md'), 'utf8'),
             'transition patch body\n',
             'transition.md content must match source byte-for-byte',
           );
         });

         it('did not write outside the sandbox HOME', () => {
           // Sanity check — nothing under real ~/.claude should be touched.
           // We can't assert the whole real HOME, but we can at least confirm
           // the function respected HOME override by checking the destination
           // was under fakeHome and not elsewhere.
           const destDir = join(fakeHome, '.claude', 'gsd-local-patches');
           const realPath = join(fakeHome, '.claude');
           assert.ok(destDir.startsWith(realPath), 'destination must be under fake HOME');
         });
       });
       ```

    2. Run it: `node --test tests/install-patches-copy.test.mjs`.
