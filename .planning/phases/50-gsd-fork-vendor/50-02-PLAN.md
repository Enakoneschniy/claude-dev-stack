**1. Rewrite `lib/install/gsd.mjs`:**

Replace the entire module. The new `installGSD()` function:
- Accepts `(stepNum, totalSteps, pkgRoot)` — same signature as before
- Uses `cpSync` from `vendor/cds-workflow/` (inside pkgRoot) to `~/.claude/cds-workflow/`
- Compares VERSION files to determine if update needed (not package.json — vendor has no package.json)
- Copies agents from `vendor/cds-workflow/agents/` to `~/.claude/agents/` (individual gsd-*.md files, NOT into `~/.claude/cds-workflow/agents/` — per Pitfall 2)
- Copies skills from `vendor/cds-workflow/skills/` to `~/.claude/skills/` (recursive copy of gsd-* directories)
- Displays step banner: `step(stepNum, totalSteps, 'Installing CDS Workflow Engine')`
- Shows version: `ok('CDS workflow engine: up to date (v${installed})')` or `ok('CDS workflow engine installed (v${bundled}')`

The `applyShippedPatches()` function: keep the export but make it a no-op that returns `{ applied: [], skipped: [], failed: [] }`. This preserves backward compatibility for any code that imports it, while dissolving the patches mechanism. Add a comment: `// Phase 50: patches mechanism dissolved — CDS owns the source now`.

New helper functions:
```javascript
function _installedCdsWorkflowVersion(dest) {
  const versionPath = join(dest, 'VERSION');
  return existsSync(versionPath) ? readFileSync(versionPath, 'utf8').trim() : null;
}

function _bundledVersion(vendorSrc) {
  const versionPath = join(vendorSrc, 'VERSION');
  return existsSync(versionPath) ? readFileSync(versionPath, 'utf8').trim() : 'unknown';
}
```

```javascript
if (hasGsd) {
  info('Updating CDS workflow engine...');
  const vendorSrc = join(PKG_ROOT, 'vendor', 'cds-workflow');
  const dest = join(homedir(), '.claude', 'cds-workflow');
  if (existsSync(vendorSrc)) {
    cpSync(vendorSrc, dest, { recursive: true });
    // Also update agents
    const agentsSrc = join(vendorSrc, 'agents');
    if (existsSync(agentsSrc)) {
      const agentsDest = join(homedir(), '.claude', 'agents');
      mkdirp(agentsDest);
      for (const f of readdirSync(agentsSrc).filter(f => f.startsWith('gsd-'))) {
        cpSync(join(agentsSrc, f), join(agentsDest, f));
      }
    }
    // Also update skills
    const skillsSrc = join(vendorSrc, 'skills');
    if (existsSync(skillsSrc)) {
      const skillsDest = join(homedir(), '.claude', 'skills');
      mkdirp(skillsDest);
      cpSync(skillsSrc, skillsDest, { recursive: true });
    }
    ok('CDS workflow engine updated');
  } else {
    warn('vendor/cds-workflow not found — skipping workflow update');
  }
}
```

Also update the detection of hasGsd (line 204) — currently checks for `gsd-` prefixed skills. Also check for `~/.claude/cds-workflow/`:
