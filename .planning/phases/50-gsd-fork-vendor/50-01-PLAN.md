  <action>
Create a one-time Node.js script `scripts/vendor-gsd.mjs` that performs the copy and path rewrite. Then run it.

The script must:

1. **Copy GSD core** — `cpSync(join(homedir(), '.claude', 'get-shit-done'), join(PKG_ROOT, 'vendor', 'cds-workflow'), { recursive: true })`

2. **Copy agents** — Only `gsd-*.md` files from `~/.claude/agents/` into `vendor/cds-workflow/agents/`:
   ```js
   const agentsSrc = join(homedir(), '.claude', 'agents');
   const agentsDest = join(PKG_ROOT, 'vendor', 'cds-workflow', 'agents');
   mkdirSync(agentsDest, { recursive: true });
   for (const f of readdirSync(agentsSrc).filter(f => f.startsWith('gsd-'))) {
     cpSync(join(agentsSrc, f), join(agentsDest, f));
   }
   ```

3. **Copy skills** — Only `gsd-*/` directories from `~/.claude/skills/` into `vendor/cds-workflow/skills/`:
   ```js
   const skillsSrc = join(homedir(), '.claude', 'skills');
   const skillsDest = join(PKG_ROOT, 'vendor', 'cds-workflow', 'skills');
   mkdirSync(skillsDest, { recursive: true });
   for (const d of readdirSync(skillsSrc).filter(d => d.startsWith('gsd-'))) {
     cpSync(join(skillsSrc, d), join(skillsDest, d), { recursive: true });
   }
   ```

4. **Bulk path rewrite** — Recursively walk all files in `vendor/cds-workflow/`. For every file with extension `.md`, `.cjs`, `.json`, `.sh`, or no extension (like `VERSION`):
   - Replace all occurrences of `/.claude/get-shit-done` with `/.claude/cds-workflow`
   - This pattern is SAFE: it only matches filesystem path references, NOT the npm package name `get-shit-done-cc`
   - Use `content.replaceAll('/.claude/get-shit-done', '/.claude/cds-workflow')`

5. **Verify no corruption** — After rewrite, grep `vendor/cds-workflow/` for `cds-workflow-cc`. If any match found, the replacement was wrong (it should be `get-shit-done-cc` for npm references). Abort with error.

6. **Set VERSION** — Write `1.36.0-cds.1` to `vendor/cds-workflow/VERSION` (per D-06 and specifics in CONTEXT.md).

7. **Remove node_modules, package.json, package-lock.json** from vendor/cds-workflow/ if they exist (they're GSD npm artifacts, not needed in vendor).

8. **Remove .git** directory from vendor/cds-workflow/ if it exists (we don't want nested git).

After writing the script, run it: `node scripts/vendor-gsd.mjs`

Then verify with: `grep -r '/.claude/get-shit-done' vendor/cds-workflow/ | head -5` — must return empty.
And: `grep -r 'cds-workflow-cc' vendor/cds-workflow/ | head -5` — must return empty.
And: `cat vendor/cds-workflow/VERSION` — must read `1.36.0-cds.1`.
