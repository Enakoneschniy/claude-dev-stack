**Step 3: Modify lib/doctor.mjs (GREEN phase)**

In the NotebookLM section (after the existing auth check and last-sync blocks, around line 119), add:

a) **Per-project stats block (D-08):**
```javascript
// Per-project notebook stats (NBLM-V2-09)
if (vaultPath) {
  try {
    const { listNotebooks, listSources } = await import('./notebooklm.mjs');
    const allNotebooks = await listNotebooks();
    const cdsNotebooks = allNotebooks.filter(nb => nb.title.startsWith('cds__'));
    
    if (cdsNotebooks.length === 0) {
      info('per-project notebooks: 0 (run notebooklm sync to create)');
    } else {
      let totalSources = 0;
      const breakdown = [];
      for (const nb of cdsNotebooks) {
        const slug = nb.title.slice(5); // strip 'cds__'
        try {
          const sources = await listSources(nb.id);
          const count = sources.length;
          totalSources += count;
          breakdown.push(`${slug}: ${count}`);
        } catch {
          breakdown.push(`${slug}: ? (error)`);
        }
      }
      ok(`${cdsNotebooks.length} notebook${cdsNotebooks.length === 1 ? '' : 's'}, ${totalSources} source${totalSources === 1 ? '' : 's'} total`);
      for (const line of breakdown) {
        info(`  ${line}`);
      }
    }
  } catch {
    // listNotebooks or listSources failed — don't block doctor
    info('per-project stats: unavailable');
  }
}
```

b) **Deprecation warning (D-09):**
```javascript
// NOTEBOOKLM_NOTEBOOK_NAME deprecation (NBLM-V2-08)
if (process.env.NOTEBOOKLM_NOTEBOOK_NAME) {
  warn('NOTEBOOKLM_NOTEBOOK_NAME is deprecated. Per-project notebooks (cds__{slug}) are now used. Will be removed in v1.0.');
  warnings++;
}
```

Place the deprecation check right after the per-project stats block, still within the `if (hasNotebooklm)` gate.

IMPORTANT: The per-project stats block calls `listNotebooks()` and `listSources()` which spawn the notebooklm CLI. These are async calls. The existing doctor function is `async function main()` so await works. But the calls add latency (~1s per CLI invocation). This is acceptable for doctor (not a hot path).
