| T-29-09 | Elevation of Privilege | chmod 0755 on copied hook — could escalate a file placed by another process | accept | Same risk profile as existing Hook 1–5 copies. `~/.claude/hooks/` is user-owned, and the source is inside the package (pkgRoot), not user-controlled. |
| T-29-10 | Repudiation | Two parallel wizard runs race on settings.json write | accept | Out of scope — wizard is an interactive single-process flow; no concurrent-run contract. Existing Hooks 1–7 share the same assumption. |
